# 📝 Online Quiz System

A simple and responsive Online Quiz System built using **HTML, CSS, and JavaScript**. 
This project is designed for teachers to conduct basic MCQ exams and for students to practice online quizzes.

## 🚀 Features

- Student Name Input
- 5 Multiple Choice Questions
- Instant Score Calculation
- Correct & Wrong Answer Count
- Responsive Design
- Simple and User-Friendly Interface
- Built with Pure HTML, CSS, and JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📁 Project Structure

```
Online-Quiz-System/
│── index.html
│── style.css
│── script.js
└── README.md
```

## ▶️ How to Run

1. Download or Clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. Enter your name.
5. Answer all questions.
6. Click **Submit Quiz** to see your result.
